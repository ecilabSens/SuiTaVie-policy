# Politique de confidentialité

Cette application utilise la caméra de votre appareil uniquement pour permettre à l'utilisateur de prendre des photos ou vidéos. Aucune donnée personnelle, image ou vidéo n'est collectée, stockée ou partagée.

Aucune information n’est transmise à des serveurs tiers. L’usage de la caméra est local à l’appareil.
